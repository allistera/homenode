# homenode

A home dashboard built with Vue 3 and Vite.

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Lint

```bash
bun run lint        # check for issues
bun run lint:fix    # auto-fix issues
```

CI runs the linter on every push and pull request to `main` via GitHub Actions.
